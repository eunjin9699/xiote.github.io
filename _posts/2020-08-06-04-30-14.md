```
C-l dt   : delete to

C-g     : go, easymotion char

C-l C-l : easymotion line
  
C-u     : undo

C-i w <cr> : save

C-k     : delete rest of line


:bq     : 버퍼 닫기

:qq     : vim 닫기

C-x C-x : recenter

C-v     : Visual

gc      : Visual Comment
```
