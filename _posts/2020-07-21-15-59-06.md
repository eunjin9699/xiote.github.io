```
C-x u
  # (undo)
C-w
  # (kill-region), cut
M-w
  # copy
C-y
  # yank, paste
  
C-d
  # (delete-char).
C-k
  # (kill-line). 
M-d
  # (kill-word). 
  
M-z
  # 해당 글자까지(포함) 지우기 (zap-to-char).
```
